Left positioned off-canvas element with Psuedo :target CSS only experience.

This pattern does not gracefully degrade under IE8 due to the :target psuedo class feature not being present in that browser. In this case, JavaScript will be required to trigger the effect or a polyfill will be needed to make the :target psuedo class compatible with IE 8. If neither can be done, then this pattern should not be used for IE 8.
