For Desktop Screens

We can apply the :hover pseudo class to activate the sub menu. However, this will not work on touch screen devices. We can apply the ".sub-menu" class to any element. Does not have to be a <div>.

For Touch Screens

We can apply a pattern modifier ".sub-menu--active" to activate the sub menu. This can be used in conjuction with JavaScript click and or touch events to apply the ".sub-menu--active" class to the sub menu element.
